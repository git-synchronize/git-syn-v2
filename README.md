# Git Synchronize

_Event-driven git remote repository syncing._

## Dependencies

- [check](https://libcheck.github.io/check)
- [libgit2](https://libgit2.org)
- [libuv](https://libuv.org)

## Compiling

```sh
make
```

## Installation

```sh
sudo make install
```

## Usage

```sh
git syn -h
Usage:
        git-syn [option] ... [command] ...
	Event-driven git remote repository syncing.
```
